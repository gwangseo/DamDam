# 📝 App Description:

담담은 친구, 연인, 가족, 혹은 처음 만난 사람과의 대화를 더 깊고 의미 있게 만들어주는 AI 기반 대화 지원 서비스입니다.
표면적인 대화에서 벗어나, 서로의 생각과 감정을 이해하고 공감할 수 있도록 돕습니다.

▶ AI 질문 카드 생성
사용자가 선택한 관계, 대화 주제, 키워드를 기반으로 인공지능이 맞춤형 질문을 생성합니다.
매번 새로운 질문 카드가 제공되어, 같은 주제라도 다른 관점에서 이야기를 나눌 수 있습니다.

▶ 4단계 심화 대화 구조
단순한 질문 한 번으로 끝나지 않고, 질문을 최대 4단계의 깊이로 확장할 수 있습니다.
단계별 질문은 점차 감정, 가치관, 경험으로 이어지며, 대화가 자연스럽게 깊어지도록 설계되었습니다.
사용자는 상하 스크롤로 질문의 깊이를 확장하고, 좌우 스크롤로 새로운 질문을 탐색할 수 있습니다.

▶ 대화 리포트 & 피드백
대화 종료 후, AI가 대화 내용을 분석해 핵심 포인트와 요약 코멘트를 제공합니다.
대화 리포트는 카카오톡이나 인스타그램 스토리로 손쉽게 공유할 수 있습니다.

▶ 맞춤형 온보딩
사용자 개개인의 대화 성향, 스타일, 관심 키워드를 설정하면, 이에 맞춰 대화 경험이 개인화됩니다.

▶ 이런 분들에게 추천합니다
친구, 연인, 가족과의 관계를 더 깊이 있게 만들고 싶은 분
의미 있는 대화를 시작하고 싶은 분
교육, 소모임, 커뮤니티 등에서 진정성 있는 대화를 이끌고 싶은 리더

담담과 함께라면, 오늘의 대화가 내일의 관계로 이어집니다.
지금, 진짜 대화를 시작해보세요. 🌿
